# InternTrack

InternTrack is a web-based platform for managing internships, enabling students to apply, track progress, and showcase skills while helping admins, faculty, and companies streamline recruitment and monitoring.

## 🚀 Features
- Student internship application & tracking
- Admin dashboard for approvals and analytics
- Company portal for posting internships
- Faculty monitoring of student progress
- Certificate upload and verification
- Role-based login system (Student, Admin, Company, Faculty)
